# Carry Trade · CIP Calculator

Ferramenta educacional para análise de carry trade via Paridade Coberta de Juros (CIP).

## Features
- Câmbio spot em tempo real (open.er-api.com, Frankfurter, jsDelivr)
- Inflação dinâmica via World Bank API
- Break-even cambial e margem de segurança do carry
- Backtest automático com 25 anos de dados (1999-2025)
- Histórico cambial interativo com drag-to-compare

## Stack
HTML/CSS/JS puro · Chart.js · APIs públicas sem autenticação
